# LED8X8 Verilog Game Project

## 專案簡介
這是一個使用 Verilog/SystemVerilog 撰寫的 8x8 LED Matrix 小遊戲專案，頂層模組為 `LED8X8`。玩家可透過左右按鍵控制角色移動，並使用射擊按鍵消除由上方掉落的物件。專案同時整合了：

- 8x8 RGB LED Matrix 顯示
- 七段顯示器分時掃描
- 按鍵控制（左移、右移、射擊、重置）
- 掉落物生成與移動
- 子彈發射與碰撞判定
- 分數 / 時間 / 等級機制
- 背景蜂鳴器音樂

整體上可視為一個以 FPGA 開發板為平台的簡易射擊 / 躲避遊戲。

---

## 頂層模組
```verilog
module LED8X8(
    output reg [7:0] DATA_R, DATA_G, DATA_B,
    output reg [3:0] COMM,
    output reg [6:0] seg,
    output reg [3:0] seg_cnt,
    input left, right, reset, shoot,
    input CLK,
    output beep
);
```

---

## I/O 說明

### Input
- `CLK`：系統時脈輸入
- `left`：角色左移按鍵
- `right`：角色右移按鍵
- `shoot`：射擊按鍵
- `reset`：遊戲重置

### Output
- `DATA_R`：8x8 LED Matrix 紅色通道資料
- `DATA_G`：8x8 LED Matrix 綠色通道資料
- `DATA_B`：8x8 LED Matrix 藍色通道資料
- `COMM`：8x8 LED Matrix 行列掃描控制
- `seg`：七段顯示器段訊號
- `seg_cnt`：七段顯示器位元選擇
- `beep`：蜂鳴器輸出

---

## 遊戲玩法
1. 玩家控制角色在 8x8 LED Matrix 底部左右移動。
2. 上方會持續生成掉落物，並往下移動。
3. 按下 `shoot` 可從角色目前位置向上發射子彈。
4. 子彈擊中掉落物時，該掉落物會被消除，並增加分數。
5. 若掉落物落到底部且撞到角色，則會扣分或造成懲罰。
6. 遊戲具有等級機制，隨著條件達成，掉落物生成頻率與難度會提高。
7. 遊戲過程中會透過七段顯示器顯示計數資訊，並透過蜂鳴器播放背景音樂。

## 顯示邏輯說明

### 8x8 LED Matrix
- `DATA_B`：顯示玩家角色
- `DATA_R`：顯示掉落物
- `DATA_G`：顯示子彈
- `COMM`：控制掃描列 / 行以完成動態顯示

### 七段顯示器
程式中使用分時掃描方式輪流更新四位數七段顯示器。從變數命名來看：
- `C1_count`, `C2_count`：分數
- `C3_count`, `C4_count`：時間或計時資訊

## 主要功能模組

### 1. 玩家控制
- 使用 `left` / `right` 控制角色在 8 個位置上移動
- `position` 紀錄角色當前位置
- `X0 ~ X7` 參數對應角色在不同欄位的顯示樣式

### 2. 掉落物系統
- 使用 `a ~ h` 紀錄 8 個欄位的掉落物位置
- `a_move ~ h_move` 控制掉落物在 LED Matrix 上的顯示
- 透過亂數 / 計數邏輯決定生成位置與掉落時機
- `level` 會影響生成頻率，等級越高掉落物越密集

### 3. 射擊系統
- `shoot` 觸發子彈發射
- `a_shoot ~ h_shoot` 對應各欄位子彈的移動計數
- `a_s_move ~ h_s_move` 控制子彈在畫面上的顯示
- 角色所在位置會轉換成對應欄位索引 `s_index`

### 4. 碰撞判定
- 使用 `a_reset ~ h_reset` 紀錄各欄位碰撞重置狀態
- 若子彈與掉落物相撞，對應掉落物會消失並計分
- 若掉落物到達玩家位置，則會觸發扣分 / 生命影響

### 5. 計分與等級
- 擊中掉落物會增加分數
- 玩家表現會影響 `level`
- 不同 `level` 下，掉落物生成規則不同

### 6. 背景音樂
- `song` 模組負責輸出 `beep`
- 內建固定旋律表，透過切換頻率播放背景音樂

## 時脈分頻模組
專案包含多個分頻器模組，用來產生不同功能所需的更新速度：

- `divfreq`：畫面更新
- `divfreq_x`：角色移動檢測
- `divfreq_1`：掉落物更新
- `divfreq_rand`：隨機生成節奏
- `divfreq_shoot`：子彈移動
- `divfreq_2`：計時器 / counter
- `divfreq_4`：射擊偵測
- `song`：蜂鳴器背景音樂

這些模組都以 `CLK` 為輸入，輸出較慢的功能時脈供不同 always block 使用。

## 使用方式（建議）
1. 將 `project.v` 匯入 FPGA 開發工具（如 Quartus / Vivado，依實際板子而定）
2. 設定 `LED8X8` 為頂層模組
3. 根據開發板實際接腳分配：
   - LED Matrix RGB 資料腳位
   - Matrix 掃描控制腳位
   - 七段顯示器腳位
   - 按鍵輸入
   - 蜂鳴器輸出
4. 完成 Pin Assignment 後編譯並下載到 FPGA
5. 使用按鍵進行遊戲操作
